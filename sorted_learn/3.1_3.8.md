+ 二分查找为例
    + 对于有序序列，才能使用二分查找法（排序的作用）
    + 维护循环不变量[l...r]
```cgo
int  binarySearch(T arr[],int n,T target){
    int l=0,r = n-1;
    while(l<=r){
        int mid = (l+r)/2;
        if(arr[mid]==target){
            return mid;
        }
        if(target > arr[mid]){
            l=mid+1;
        }else{
            r = mid-1;
        }
    }
    return -1;
}
```
